# Игра "Найди пару"

Простая игра, построенная на React: 4x4 доска с 8 уникальными парами цветов. Можно кликать на две плитки - если пара, плитки переворачиваются обратно. Выигрыш когда все открыты. Имеется тёмная тема, а также Tailwind стили.

## Технологии
- React + Vite
- Tailwind CSS

## Запуск в dev-режиме
1. Клонируйте репозиторий: `git clone https://github.com/[твой-username]/findthepair.git`
2. Установите: `npm install`
3. Запустите: `npm run dev` — откройте локальный сервер

## Деплой
Развернуто на: https://[твой-username].github.io/findthepair
